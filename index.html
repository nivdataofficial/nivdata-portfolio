Real Estate Data Cleanup Project

This case study demonstrates the transformation of a NYC property sales dataset from a corrupted, unreadable export into a production-ready, analysis-grade dataset. The tool identified and resolved 6 critical data integrity issues across 1,200+ property records while maintaining full traceability and 100% data fidelity.

Issues Identified

  • Sale Date displaying as '########' from column overflow
  • Building Class Category truncated ("07 RENTAL" instead of full name)
  • Sale Price corrupted with placeholder values ($1, $10)

  • Missing values in Apartment Number and Square Footage fields
  • Duplicate records for the same unit across multiple rows

Key Improvements

  • Building Class recovered to full descriptive labels
